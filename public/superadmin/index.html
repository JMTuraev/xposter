🧪 Центр тестирования
Быстрые проверки (~4 с): инфраструктура, целостность данных всех заведений, подписки, интеграционный поток, модель ролей.

Полный E2E тест (~30–60 с): создаёт тест-аккаунты для каждой роли и по-настоящему вызывает КАЖДУЮ Cloud Function (createEmployee, setEmployeeActive, completePayment, deleteAccount…) и проверяет ВСЕ правила доступа по ролям (официант/повар/менеджер/чужой) + подписку. В конце всё удаляется.
