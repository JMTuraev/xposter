# PROMPT 2/5 — Касса (POS-terminal)

---

Продолжаем прототип «Buxoro POS». Не меняя дизайн-систему и уже готовые экраны, полностью реализуй вкладку **«Касса»** — мобильный POS-терминал (аналог кассы Poster). Всё интерактивно на mock-данных.

## 1. PIN-экран (при первом входе на вкладку)
- Логотип «Buxoro POS», подпись «Больше, чем просто касса».
- 4 точки-индикатора, цифровая клавиатура 1-9, «Удалить», 0, ⌫.
- Внизу: название заведения «Чайхана „Бухоро"», «Касса №1», клиентский номер 567053.
- PIN 0000 → Жафар (Владелец), 1111 → Азиз, 2222 → Малика. Неверный PIN — тряска точек + красный цвет.
- После входа имя кассира показывается в шапке кассы; иконка замка 🔒 снова блокирует кассу (смена кассира за 2 сек).

## 2. Главный экран кассы (экран «Чек»)
Верхняя тёмная панель (исключение из светлой темы, как у Poster: #2D2C28):
- «‹ Заказы» (список заказов), табы «Чек | Клиент», по центру «Чек №1 ▾» (dropdown: список открытых чеков + «Создать новый» — параллельные чеки), справа: 🗃 (архив), ☰ (функции), имя кассира 🔒, зелёная точка «онлайн».

Тело (мобильная адаптация: две зоны вертикально):
- **Каталог**: строка «Все товары › [категория]» (breadcrumb, тап назад), иконки: штрих-код, 🔍 поиск (живой фильтр по названию), кнопка «Акции».
  Плитки категорий (эмодзи + название), тап → плитки товаров (эмодзи, название, цена). Тап по товару добавляет позицию в чек (повторный тап +1). У «Вода минеральная» — bottom sheet выбора модификации (Газированная/Без газа).
- **Панель чека** (снизу, поднимается свайпом/тапом): список позиций «Название | Кол-во | Цена | Итого», тап по позиции → bottom sheet позиции: Комментарий (textarea), Количество [−1][−0,5][ 2 ][+0,5][+1], «Удалить позицию» красным.
  Под списком: «К оплате: 105 000 СУМ» (серифный крупный), кнопки [⋯] (меню: «Комментарий к чеку…», «Очистить заказ» с подтверждением), [🖨], зелёная **[Оплатить]**.

## 3. Вкладка «Клиент» (в шапке кассы)
- Поиск по имени/телефону, список клиентов из MOCK (имя, телефон, группа, % скидки/бонусы).
- Тап — клиент прикреплён к чеку: в панели чека появляется чип «Азиз Каримов · скидка 5%», сумма пересчитывается; у VIP видно «Бонусы: 240 000, можно оплатить до 50% чека».
- Пустой поиск: «Здесь будут клиенты…» + [Новый клиент] → bottom sheet: ФИО, Пол (Мужской|Женский), Дата рождения, Группа (select), Номер карты, Персональная скидка %, Телефон (+998 префикс), E-mail, Комментарий, Адрес (Город/Улица/Дом) → «Добавить» создаёт клиента в MOCK + тост.

## 4. Экран оплаты (по «Оплатить»)
- Шапка: «‹ Отменить», «Чек №1».
- Слева/сверху numpad: быстрые суммы [105 000] [110 000] (точная и округлённая), цифры, «.», ⌫.
- «К оплате: 105 000 СУМ». Способы оплаты — строки-кнопки: 💵 Наличными, 💳 Карточкой (сплит: тап по строке вносит остаток; можно вручную с numpad разделить), кнопка «Сертификатом», «Бонусами» (если клиент VIP — списывает бонусы, максимум 50%).
- Тумблер «Напечатать чек». Красная ссылка «Закрыть без оплаты» (подтверждение). Зелёная [Оплатить].
- Валидация как в Poster: если сумма не набрана — строки способов подсвечиваются красной рамкой.
- Наличными больше суммы → показать «Сдача: 5 000 СУМ».
- После оплаты: полноэкранный чек-превью (бумажный стиль: название заведения, адрес, кассир, позиции, итого, способ оплаты, «Спасибо! Рахмат!»), кнопки «Новый чек» и «Поделиться». Заказ уходит в Архив, статистика Главной обновляется (+1 чек, +выручка!), остатки ингредиентов списываются по тех. карте плова.

## 5. Экран «Заказы» («‹ Заказы» из шапки)
- Табы «Заказы | Архив чеков», зелёная [Новый заказ], поиск.
- Заказы: карточки открытых заказов: время «00:06», «№12 › В ЗАВЕДЕНИИ · Стол 3», позиции кратко, сумма, синяя кнопка [Перейти к оплате].
- Архив: фильтры-чипы «Все чеки | Наличными | Карточкой | Возвраты» + «Сегодня ▾»; список «№11 — 148 000 СУМ — Плов ×2, Шашлык…»; тап → детальный чек: Кассир, Открыт/Закрыт время, таблица позиций, Итого, Оплата; кнопки [Чек] (печать) и [Возврат] (подтверждение → чек в «Возвраты», статистика минусуется).

## 6. ☰ «Функции» (модал)
Сетка карточек по группам:
- Приложения: 📋 Инвентаризация, 📦 Поставка, ✨ «Накладная с фото (AI)» — открывают соответствующие формы Склада (шаг 3; пока тост «Появится в модуле Склад»).
- Оборудование: 🖩 Устройства, 💰 «Открыть денежный ящик» (тост «Ящик открыт»).
- Другое: 📄 «Составить отчёт» → bottom sheet: «Начиная с» (дата+время), «Заканчивая», «Кассир» (select «Все кассиры»), тумблер «Продажи по товарам», [Составить отчет] → экран X-отчёта (выручка по способам оплаты, чеки, средний чек за период); ⠿ «Режим сортировки» (плитки дрожат, можно перетаскивать — достаточно имитации), 🔄 «Очистить кеш» (тост), 🚪 «Выйти из аккаунта» → PIN-экран.

Проверь перед сдачей: добавление в чек, изменение количества, скидка клиента, сплит-оплата, сдача, возврат, параллельные чеки, поиск — всё работает без ошибок в консоли.
